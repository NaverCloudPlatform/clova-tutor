# 주요 기능

CLOVA Tutor는 **수학**과 **영어** 두 과목의 AI 튜터를 통해, 하나의 채팅 인터페이스 안에서 **개념 설명·문제 추천·단계적 풀이·직독직해**를 경험할 수 있습니다. 정적인 콘텐츠가 아닌 **실시간 대화**를 바탕으로, **학습 질의응답 → 추천 문제 풀이 → 관련 질문 → 복습**으로 이어지는 학습 사이클을 하나의 흐름으로 이어갈 수 있도록 설계되었습니다.

---

## 수학 튜터

수학 튜터와의 대화를 통해 개념 설명, 맞춤 문제 추천, 단계별 풀이와 해설을 받을 수 있습니다. 학습자의 반응에 따라 설명 방식과 난이도를 조절하는 상호작용을 통해, 스스로 문제를 찾고 해결해 나가는 **자기주도 학습**을 지원합니다.

### 개념 설명과 문제 추천

이해가 필요한 개념을 질문하면 실시간으로 설명을 받고, 이어서 **수준에 맞는 문제**를 추천받을 수 있습니다. '정답 제공'에 그치지 않고, 학습자가 스스로 다음 단계를 이어갈 수 있도록 **추천 문제 풀이 → 관련 질문**으로 자연스럽게 연결됩니다.

## 영어 튜터

영어 튜터와의 대화를 통해 단어·문법 설명, 직독직해, 문법/문제 추천을 이어갈 수 있습니다. 학습자의 반응에 맞춰 설명 방식과 난이도를 조절하여, 혼자 공부할 때의 부담을 줄이면서 **자연어 기반 대화**로 이해도를 높일 수 있습니다.

### 단어 설명과 직독직해

모르는 단어나 문장을 질문하면 뜻과 쓰임을 설명받고, 문장 단위 **직독직해**를 연습할 수 있습니다. 정적인 자료가 아닌 대화를 통해 궁금한 부분을 바로바로 질문하고, 그에 맞는 설명을 받으며 학습을 이어갈 수 있습니다.

## 학습 관리

학습 목표와 저장된 노트를 통해 **학습 지속성**을 확보하고, 문제 풀이와 복습이 분리되지 않는 **연결된 학습 흐름**을 경험할 수 있습니다.

- **학습 목표 설정**: 채팅방에서 목표를 정하면 튜터가 그에 맞는 **피드백과 가이드**를 제공합니다. 목표 기반으로 대화가 이어져, 스스로 학습을 이어가는 데 도움이 됩니다.
- **학습 노트**: 풀었던 문제와 해설을 노트로 저장해 나중에 다시 복습할 수 있습니다. 저장된 문제는 **다시 풀기**가 가능하고, 해당 문제의 **원본 채팅으로 이동**하여 이전 대화 맥락을 이어갈 수 있습니다.
- **연결된 학습 흐름**: 채팅 이력과 학습 노트가 연동되어 이전 맥락을 유지하며, 재풀이와 복습이 자연스럽게 이어집니다. 학부모나 교사를 대체하지 않는 **보조적 학습 파트너**로서, 혼자 공부할 때의 부담을 줄이는 데 초점을 둡니다.

자세한 사용 방법은 [사용자 가이드](/docs/user-guide/start)를 참고하세요.
