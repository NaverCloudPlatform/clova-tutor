# Getting Started

CLOVA Tutor is a demo project designed for exploration and experimentation, and it can be used without a formal sign-up process.  
However, to support grade-specific learning experiences, basic user identification is required. For this reason, you need to create an account when using the service for the first time.

---

## Creating a student account

![Create Student Account](@site/docs/user-guide/img/학생계정생성.gif)

When you first access the site, a message appears indicating that no account has been created yet.  
Click **Create account** to generate a new student account.

---

## Logging in

![Login](@site/docs/user-guide/img/로그인.gif)

Select an existing account name to log in and start using CLOVA Tutor.

---

## Logging out

![Logout](@site/docs/user-guide/img/로그아웃.gif)

Click the user icon in the sidebar and select **Log out** to sign out.  
You will be redirected to the account management page, where additional accounts can be created.
