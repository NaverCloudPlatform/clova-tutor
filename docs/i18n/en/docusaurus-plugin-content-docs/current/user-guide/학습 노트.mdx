# Study Notes

## Accessing the Study Notes page

The **Study Notes** page can be accessed from the sidebar by selecting **Study Notes**.

## Solving problems

Selecting a problem from the Study Notes list opens a dialog where the problem can be solved again.

<div className="flex flex-col gap-4 divide-x">
  <img src={require('@site/docs/user-guide/img/학습노트단일문제.png').default} className="px-4" />
  <div>

<p className="bg-accent-blue-400 size-5 rounded-full flex items-center justify-center text-white font-semibold">1</p>
> Grade level, problem difficulty, and problem category information are displayed.

<p className="bg-accent-blue-400 size-5 rounded-full flex items-center justify-center text-white font-semibold">2</p>
> The saved problem is shown. Problems may be multiple-choice or short-answer.

<p className="bg-accent-blue-400 size-5 rounded-full flex items-center justify-center text-white font-semibold">3</p>
> Closing the dialog returns you to the saved problem list.

<p className="bg-accent-blue-400 size-5 rounded-full flex items-center justify-center text-white font-semibold">4</p>
> Arrow buttons allow navigation between previous and next problems.

<p className="bg-accent-blue-400 size-5 rounded-full flex items-center justify-center text-white font-semibold">5</p>
> Use **Ask in Chat** to ask the AI about the current problem, or **Grade** to check your answer.

  </div>
</div>

## Removing problems from Study Notes

<img src={require('@site/docs/user-guide/img/학습노트저장해제.gif').default} />

- Selecting the bookmark button removes a problem from Study Notes.
- To prevent accidental removal, the problem is not immediately deleted from the list. Only the bookmark toggle is turned off.
- The problem can be re-added at any time by toggling the bookmark again. The removal is finalized after a page refresh.
