## <div className="inline-flex items-center "><span className="bg-accent-blue-400 size-[1.3em] text-[0.7em] rounded-full inline-flex items-center justify-center text-white font-semibold">4</span>&nbsp;Opening and closing the problem panel</div>

- When a tutor response includes recommended problems, a summary panel is generated automatically.
- Click **Open** to view the problem at any time.
- Use the **Open / Close** button to show or hide the learning problem recommendation panel.
- The chat layout can be adjusted freely while the panel is open.
