### Attaching problem images

<img src={require('@site/docs/user-guide/img/이미지첨부.png').default} width={50} />

- You can upload up to one problem image.
- Image requirements:
  - Aspect ratio between 1:5 and 5:1
  - File size under 20MB
  - Longest side up to 2240px, shortest side at least 4px
  - Supported formats: jpeg, jpg, png
- The tutor analyzes the image and responds to your question.
- Images copied to the clipboard can be pasted using `Ctrl + V`.

### Inserting math input blocks

<img src={require('@site/docs/user-guide/math-tutor/img/수식삽입.png').default} width={50} />

Click the math input block button to activate the math keyboard and enter mathematical expressions.
