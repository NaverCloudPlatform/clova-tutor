### Attaching problem images

<img src={require('@site/docs/user-guide/img/이미지첨부.png').default} width={50} />

- You can upload **one** problem image at a time.
- Image upload requirements:
  - Aspect ratio between **1:5 and 5:1**
  - File size up to **20 MB**
  - Longer side up to **2240 px**, shorter side at least **4 px**
  - Supported formats: **jpeg, jpg, png**
- The tutor analyzes the uploaded image and responds to your question.
- Images copied to the clipboard can be pasted using `Ctrl + V`.

- View up to **10 recent chat rooms** in the list.
- The first question sent in a chat room is used as the chat room title.
- Click **All Chats** to view the full chat history.
