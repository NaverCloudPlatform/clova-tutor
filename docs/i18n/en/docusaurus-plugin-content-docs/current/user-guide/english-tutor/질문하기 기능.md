# Asking Questions in Chat

## Chat — Select Text and Ask the AI Tutor

![Chat - Select Text and Ask the AI Tutor](@site/docs/user-guide/english-tutor/img/영어ai에게질문.gif)

- When you select a sentence or word in the conversation, an **Ask AI Tutor** button appears.
- Clicking the button automatically quotes the selected content in the chat input.
- You can then add your own question and continue the conversation freely.
